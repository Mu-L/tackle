## 目标

## 校验范围

### 1. 未修改文件的全局搜索

```bash
# 检查其余 17 个 skill.md 文件是否残留"三位数字"
grep "三位数字" plugins/core/*/skill.md
```

校验要点：
- [ ] 除 WP-057~WP-060 修改的 6 个文件外，其余文件无"三位数字"匹配
- [ ] 确认问题仅限于已识别的文件

### 2. Git Diff 全局分析

```bash
git diff --stat
```

校验要点：
- [ ] 修改文件列表仅包含预期的 6 个 skill.md 文件
- [ ] 无 JavaScript 代码文件被修改
- [ ] 无测试文件被修改
- [ ] 无配置文件被修改

### 3. Markdown 完整性检查

对 6 个修改的文件逐一检查：
- [ ] 表格格式完整（无断行）
- [ ] 代码块正确闭合
- [ ] 标题层级正确
- [ ] 无重复内容或重复章节
- [ ] 链接引用有效

### 4. 变更合理性审查

- [ ] WP-057 修改量 ~14 行（硬约束移除）
