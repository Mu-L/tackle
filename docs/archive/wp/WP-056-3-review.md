## 职责

## 报告结构

```
1. 验证概述
   - 验证方法
   - 验证范围
   - 评分体系

2. 逐项验证结果表
   | 原始结论 | 原始行号 | 验证结果 | 差异说明 |

3. 报告准确性评分
   - 按 High/Medium/Low/None 分组
   - 每组给出总体评分（准确/基本准确/有偏差/不准确）

4. 新发现（原报告未覆盖）
   - skill-split-work-package 子任务编号前导零模式
   - 其他验证中发现的遗漏

5. 修复建议评价
   - 对原报告每条修复建议的合理性评价
   - 是否有建议遗漏或过度修复

6. 总体结论
   - 原报告的可信度评级
   - 需要修正的具体条目（如有）
```
